<p>European Repair Package (ERP) is a community patch pack for the game, <a title="Steam Store" href="https://store.steampowered.com/app/793460/112_Operator/?curator_clanid=33067827" target="_blank" rel="noopener noreferrer">112 Operator</a>. This pack will consist primarily of language repair lines for the base game and Steam Workshop assets but may also include the odd item, as deemed necessary to fix it. It a patch is noticed to have correct the issue in the base game or Steam Workshop asset, it will be removed from this mod keeping this a light weight repair pack.</p>
