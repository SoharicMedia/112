<p>European Response Package (ERP) is a community modpack for the game, <a title="Steam Store" href="https://store.steampowered.com/app/793460/112_Operator/?curator_clanid=33067827" target="_blank" rel="noopener noreferrer">112 Operator</a>. Much like <a href="911/otl">On the Line</a>, our previous mod for <a title="Steam Store" href="https://store.steampowered.com/app/503560/911_Operator/" target="_blank" rel="noopener noreferrer">911 Operator</a>, ERP aims to increase the quantity of in-game content (calls, incidents, vehicles etc.) while also providing support for all base languages and extending support for further workshop languages. Call voiceovers will be available only in English, but for the list of supported text languages, please see the <a href="#languages">languages</a> tab below.</p>
